Computer Move Strategies

Easy Mode

In easy mode, the computer makes completely random moves. It will choose any available empty square without any strategy, making it perfect for beginners or casual play.

Medium Mode

Medium mode combines strategy with randomness. The computer will:

  • Try to win if it has a winning move
  • Block your winning moves
  • Sometimes make optimal moves (30% chance)
  • Otherwise make random moves

Hard Mode

Hard mode uses the Minimax algorithm, making it unbeatable. The computer will:

  • Always choose the optimal move
  • Never make mistakes
  • Either win or force a draw
